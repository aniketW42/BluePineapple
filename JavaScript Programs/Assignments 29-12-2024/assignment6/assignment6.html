    Task: 
    Create a webpage that includes a button labeled "Add Item". 
    When the button is clicked, dynamically create a new list item (`<li>`) and append it to an existing unordered list (`<ul>`). 
    The new list item should contain the text "Item X" where X is the number of items currently in the list (e.g., "Item 1", "Item 2" etc.).
